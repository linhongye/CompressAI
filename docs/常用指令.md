# 常用指令

## master plan 的执行

- @masterPlan.md 目前已经完成phase X, 请开始规划完成phase X
- 代码修改完成后的验证阶段:
    - 使用新完成的代码, 进行一次训练:
        - epoch = 100
        - batch size = 32
        - 训练时, 使用的图片在: @image/train
        - 训练后, 权重文件保存在: @image/model 
        - 训练后, 权重文件命名为: phaseX_best 和 phaseX_last
    - 使用best 权重进行一次benchmark
        - 图片用: @image/test_process 
        - report 保存在: @image/test_process /report_phaseX
